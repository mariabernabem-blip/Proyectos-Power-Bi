# Proyecto Power Bi:
## Un análisis mediante Power Bi de los videojuegos más consumidos mundialmente.

## Objetivos
En este proyecto de nivel básico de Power Bi, voy a llevar a cabo un reporte interacivo que nos permita analizas las ventas de millones de copias de videojuegos a nivel mundial.

Para ello, vamos a analizar por las categorías de: año, región, plataforma,  género y editorial. 
Finalmente se llevarán a cabo unas conclusiones acerca de este proyecto.
